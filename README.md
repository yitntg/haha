# DeepSeek Chat UI

这是一个模仿 DeepSeek Chat 界面的静态前端项目。

## 技术栈

- React 18
- TypeScript
- Tailwind CSS
- Vite
- React Icons

## 项目结构

```
src/
  ├── components/      # React 组件
  │   ├── Header.tsx  # 顶部标题组件
  │   └── ChatInput.tsx # 底部输入框组件
  ├── App.tsx         # 主应用组件
  ├── main.tsx        # 应用入口
  └── index.css       # 全局样式
```

## 功能特点

- 深色主题界面
- 响应式设计
- 模拟聊天输入框
- 功能按钮展示（深度思考、联网搜索等）
- 自定义滚动条样式

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
